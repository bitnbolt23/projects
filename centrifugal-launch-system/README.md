# Centrifugal Launch System 🚀

# Overview
This project is a hardware-based propulsion system developed during a first-year engineering competition. It demonstrates the use of centrifugal force to launch a small metal shuttle in a controlled manner.


# Objective
The primary objective of this project was to explore an alternative launch mechanism that could reduce the dependency on multi-stage fuel-based propulsion systems. By minimizing the number of fuel stages, the system aims to reduce overall weight, complexity, and cost of launch systems.


# Concept
The system consists of a rotating mechanism enclosed within a structure. A metal shuttle is placed inside and spun at high speed. At a specific moment, the shuttle is released, converting rotational motion into linear motion due to centrifugal force.


# Working Principle
- When an object rotates, it experiences an outward force (centrifugal force)
- As rotational speed increases, the force increases
- At the right time, a release mechanism allows the shuttle to exit tangentially
- This results in a projectile-like motion
